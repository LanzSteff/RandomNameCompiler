RandomName - README
=====================

Verwendung:
Mit dem Befehl 
	ant jar

kann eine ausführbare jar Datei erstellt werden, welche danach mit 
	java -jar RandomName.jar <Filename>

ausgeführt werden kann. Mit Filename ist ein Textfile gemeint, welches danach eingelesen, compiliert und ausgeführt wird. Als Ausgabe auf der Console erhält man zuerst den Programflow, gefolgt von den diversen Symboltables. Danach werden die belegten Plätze im Speicher ausgegeben, und zu guter letzt die einzelnen Instruktionen, sowie die Werte in den Registern am Ende.
